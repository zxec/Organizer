# Organizer

## What is the app?
Это приложение представляет из себя календарь с возможностью добавления заданий на каждый день. В нём будет возможно отслеживать выполнения заданий.
Данное приложение будет полезно для тайм-менеджмента.

## Technologies 
- HTML and HTML5
- CSS
- CSS Flexbox
- CSS Grid
- TypeScript
- JavaScript
- Angular
- Moment.js
- RxJS
